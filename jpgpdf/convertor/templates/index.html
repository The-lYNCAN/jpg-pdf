How to use The site?

  1. First Create a new directory and arrange the file in series of the way you want your pdf (i.e. first page you can select at wish, then the second page should be on th first place in the folder, third page second picture in the folder)
  2. Upload the first image using the "choose file" button above
  3. Press "Next" button
  4. Choose all the images you want to display in the pdf in a single click (Please watch the video) on the next page
  5. Click "Download" button to download the pdf
  6. Thank you! and Please Share the website!
